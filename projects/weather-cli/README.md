# Weather CLI Tool 

My first real Python project built from scratch.
Fetches live weather data from the terminal.

## What it does
- Fetches live weather for any city
- Shows temperature in Celsius or Fahrenheit
- Saves results to a JSON file
- Full error handling

## How to run
pip install requests

python weather.py Lahore
python weather.py Lahore --farenhiet
python weather.py Lahore --save

## Built with
- Python
- requests
- argparse
- JSON file handling

## What I learned
- API calls with error handling
- Nested JSON parsing
- CLI tools with argparse
